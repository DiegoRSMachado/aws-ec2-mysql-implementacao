# ☁️ AWS Cloud Infrastructure & MySQL Implementation

### 📝 Resumo do Projeto
Implementação de um servidor de banco de dados MySQL em instância **AWS EC2 (Ubuntu)**, com configuração de segurança de rede e acesso remoto via MySQL Workbench.

### 🔒 Segurança na Nuvem (AWS Security Groups)
Configuração de firewall de nuvem (Security Group ) permitindo apenas tráfego essencial:
* **Porta 22 (SSH):** Gestão remota do servidor Linux.
* **Porta 3306 (MySQL):** Conexão segura de banco de dados.
* **Porta 80/443:** Servidor Web.

### 🛠️ Tecnologias Utilizadas
* **Cloud:** AWS EC2.
* **Database:** MySQL Server 8.0.
* **Tooling:** MySQL Workbench para gestão visual.
* **SQL:** Scripts DDL/DML para estruturação de tabelas de RH.
